# Simax — Help-Desk Platform (Demo)

Interactive demo of an **enterprise help-desk platform**: ticketing, SLA tracking,
escalations and client management — with a strong focus on UI/UX.

🔗 **[Live demo](https://valentinopais.github.io/simax-demo/)**

https://github.com/user-attachments/assets/4e878da3-c1de-487c-b7b3-7454716ada93

## What it solves
Companies handling internal incidents need one place to **create, assign, escalate and
close tickets** while tracking SLAs and client data. Simax centralizes that workflow in a
fast, clean interface that agents actually enjoy using.

## Highlights
- **Console** — filterable/paginated incident table with live status tabs
- **Incident detail** — editable forms, events timeline, escalations, attachments
- **Eventos & Escalamientos** — modal flows with role-based visibility
- **Solicitantes** — client management with inline create/edit/delete
- **Dashboard** — interactive charts (status distribution, breakdowns)

## Stack
`JavaScript` · `HTML` · `CSS` — single-file SPA demo, no build step.
Original product built with **ASP.NET MVC + SQL Server** at Xoftech.

## Note
This is a front-end demo recreating the real product's UX. Production code is private.
